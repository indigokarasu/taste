# Taste

Behavior-driven taste model. Builds personalized recommendations from real consumption signals with evidence-backed explanations. Supports cross-domain discovery and periodic taste reports.

## Repository Structure

- `ocas-taste/` - Skill package source
- `spec/` - Specification and build documentation

## Installation

This skill package is available on GitHub. To use it, clone this repository and follow the installation instructions in your skill management system.
